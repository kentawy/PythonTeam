Питання: Що робить функція print() у Python?
Прізвище та ім'я: Петрушко Ярослав
Відповідь: Функція print() виводить вказаний об’єкт на стандартний пристрій виводу (екран) або у файл текстового потоку.
Функція print() може приймати до 5 параметрів: *objects - об'єкт для виводу; sep - роздільник, для декількох об'єктів; end - що виводити в кінці; file - об'єкт із методом запису; flush - скидання виводу (True) чи буферизація (False).
Питання для наступного студента: Як виконати зріз рядка у Python?
Прізвище та ім'я: Петрушко Ярослав
Відповідь: text
Питання: example
Прізвище та ім'я: Рубан Богдан
Відповідь: Зріз рядка в Python виконується за допомогою синтаксису рядок[start:stop:step], де start – початкова позиція зрізу (включно), stop – кінцева позиція зрізу (не включно), step – крок, з яким виконується зріз.
Приклад:
text = 'Python'
text[1:4] поверне 'yth', text[:3] поверне 'Pyt', text[::2] поверне 'Pto'.
